# Numeric String

Format any number into a string with "," (commas) in the correct places. E.g. "1,000,000".


# 숫자 포맷 다듬기

formatString(2000); => '2,000'
formatString(20000); => '20,000'
formatString(200000); => '200,000'
formatString(2000000); => '2,000,000'
